# basic-calculator
Basic calculator web app using HTML, CSS, and JavaScript. Designed with a clean UI and essential arithmetic features.


## Tech Stack
- **HTML5**  
  Digunakan untuk membangun struktur tampilan kalkulator.
  
- **CSS3**  
  Digunakan untuk mengatur tampilan antarmuka agar terlihat rapi, modern, dan mudah digunakan.
  
- **JavaScript (Vanilla JS)**
  Digunakan untuk mengatur logika perhitungan, event handling, dan interaksi pengguna.

  
---

## How to Use
1. Buka aplikasi melalui browser.
2. Masukkan angka menggunakan tombol yang tersedia.
3. Pilih operasi matematika yang diinginkan (`+`, `-`, `ร`, `รท`).
4. Tekan tombol `=` untuk melihat hasil perhitungan.
5. Gunakan tombol `C` untuk menghapus semua input.
